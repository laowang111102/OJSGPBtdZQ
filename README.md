# 项目简介 python102

本项目是基于Python和OpenCV的人脸录入与识别系统，运用dlib机器学习库提升识别效果。

## 技术栈

- **后端**：Python
- **图形界面**：Tkinter
- **人脸识别库**：OpenCV、dlib

## 功能模块

- **人脸录入界面**：使用Tkinter构建，支持录入时输入中文姓名。
- **摄像头调用**：用于实时人脸识别，支持多人同时识别。
- **姓名显示**：识别结果支持中文姓名展示。

## 系统角色

- **用户**：通过界面录入人脸信息，查看识别结果。

## 运行环境

- 操作系统：兼容Windows、Linux和Mac OS。
- Python版本：3.6及以上。

## 部署步骤

1. 安装Python环境和相关依赖库（OpenCV、dlib、Tkinter）。
2. 克隆或下载项目代码。
3. 执行项目运行脚本。

## 目录结构

```
项目根目录/
│
├── main.py           # 主程序入口
├── face_recognition  # 人脸识别模块目录
│   ├── face_detect.py  # 人脸检测代码
│   └── face_train.py   # 人脸训练代码
│
└── gui               # 图形界面模块目录
    ├── main_window.py # 主窗口代码
    └── input_name.py  # 输入姓名界面代码
```

## 核心亮点

- **支持中文**：录入和显示姓名时，兼容中文字符。
- **实时识别**：摄像头实时捕捉，快速响应。
- **多目标识别**：可同时识别多张人脸并显示姓名。
